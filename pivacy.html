Privacy Policy

Last updated: March 2026

1. Information We Collect

Daily Quote Bot does not collect personal information from end users. The App operates as an automated content publisher and does not have user registration or login features.

2. TikTok API Usage

This App connects to TikTok's API using credentials authorized by the App owner. We access only the permissions necessary to publish video content (video.upload and video.publish scopes). We do not access follower data, messages, or any personal account information beyond what is required for content publishing.

3. Data Storage

No user data is stored by this App. Generated content (quote images and videos) is stored locally on the App operator's device only.

4. Third-Party Services

This App uses the TikTok API. Please refer to TikTok's Privacy Policy at https://www.tiktok.com/legal/privacy-policy for information on how TikTok handles data.

5. Cookies

This website does not use cookies or any tracking technologies.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated date.

7. Contact

For privacy-related questions, please contact us through GitHub at https://github.com/monkichi27/tiktok-app.
